Vigenere Cipher

It is designed by Blaise de vigenere.It is an algorithm that is used to encryption and decryption the text. It is an example of polyalphabetic substitution ciphers. This algorithm is easy to understand and implement.
The encryption is done using two methods

1. vigenere table is given
Plain text: GIVE MONEY
key: LOCK

Encryption

Repeat the letters of key so that number of letters in plain text are equal to the letters in the key

Decryption

Second method Without Table

1. Encryption: Ciphered text= (Plain text + key)%26
2. Decryption: Plain text= (Ciphered text - key)%26

key: PASCAL
